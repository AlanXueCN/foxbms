========
Releases
========

The current version of |foxbms| is |version|.

.. note::

    EEPROM module from **v1.5.0** is **NOT** compatible with previous versions
    due to the changed memory layout because of added timestamps!

.. note::

    foxBMS uses now python3. The foxconda3 installer is found at
    https://iisb-foxbms.iisb.fraunhofer.de/foxbms/. foxconda3 must be installed
    to C:\\foxconda3!

.. note::

    Until version **v1.1.0**, the |foxbms| software was split into several
    repositories, with an independent version number for each repository.
    Starting from version **v1.5.0**, the |foxbms| software is contained in
    only one repository, with one version number!

The following tables describe the different versions of |foxbms| that were
released. The first line is the most recent one, the last one the oldest one.
